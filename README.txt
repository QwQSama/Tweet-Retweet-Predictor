This file describes how to use our submission code.

First, put the data set(.csv) file in the same path as the jupyter notebook(.ipynb).

And make sure the following libraries are already included in your python distribution:

	numpy

	pandas

	sklearn

	verstack

	nltk

	seaborn

	beautifulsoup4(bs4)

	regular expression operations(re)

	scipy

Then you can feel free to test our code by running the cells in jupyter notebook.

The results will be displayed below each cell.
